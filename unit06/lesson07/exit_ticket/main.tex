A quality inspector at a food packaging plant tests whether the proportion of
underweight packages exceeds 2\%. The hypotheses are $H_0: p = 0.02$ vs.\ $H_a: p > 0.02$.

\begin{enumerate}
  \item Describe a Type I error in this context.
        \writelines{2}

  \item Describe a Type II error in this context.
        \writelines{2}

  \item Which error is more serious from the perspective of consumer safety?
        \writelines{2}

  \item The plant increases the sample size from 200 to 800. How does this affect power?
        \writelines{1}
\end{enumerate}
