A researcher claims that 25\% of adults report daily physical activity. A fitness company
believes the true proportion is \emph{greater than} 25\%. From 300 randomly selected adults,
84 report daily activity. A hypothesis test yields $z \approx 1.20$.

\begin{enumerate}
  \item State the hypotheses.
        $H_0$: \blank{4cm} \quad $H_a$: \blank{4cm}

  \item Compute the $p$-value.
        Tail: \blank{2cm} \quad $p$-value $=$ \blank{4cm}

  \item Write a complete contextual interpretation of the $p$-value.
        \writelines{3}

  \item A classmate says: ``The $p$-value is 0.115, so there is an 11.5\% chance that
        $H_0$ is true.'' Identify and correct the error.
        \writelines{2}
\end{enumerate}
