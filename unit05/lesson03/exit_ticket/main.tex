Reaction times for drivers follow a right-skewed distribution with $\mu = 1.5$ seconds
and $\sigma = 0.4$ seconds. A researcher randomly selects $n = 45$ drivers.

\begin{enumerate}
  \item Are the conditions for using the normal model for $\bar{x}$ met? State and check
        each condition.
        \writelines{4}

  \item Find $\mu_{\bar{x}}$ and $\sigma_{\bar{x}}$.
        \vspace{0.6in}

  \item Find $P(\bar{x} < 1.4)$.
        \vspace{0.8in}
\end{enumerate}
