A school surveys students about after-school jobs. Among 11th graders: $p_1 = 0.35$,
$n_1 = 100$. Among 12th graders: $p_2 = 0.52$, $n_2 = 80$.

\begin{enumerate}
  \item Check the four Large Counts conditions.
        \vspace{0.6in}

  \item Find $\mu_{\hat{p}_1-\hat{p}_2}$ and $\sigma_{\hat{p}_1-\hat{p}_2}$.
        \vspace{0.6in}

  \item Find $P(\hat{p}_1 - \hat{p}_2 < -0.25)$.
        \vspace{0.9in}
\end{enumerate}
