Two groups of employees attend different training programs. Program 1 employees:
$\mu_1 = 88$ (productivity score), $\sigma_1 = 10$, $n_1 = 40$. Program 2 employees:
$\mu_2 = 82$, $\sigma_2 = 14$, $n_2 = 50$. Both populations are approximately normal.

\begin{enumerate}
  \item Check all conditions for using the normal model for $\bar{x}_1 - \bar{x}_2$.
        \writelines{3}

  \item Find $\mu_{\bar{x}_1-\bar{x}_2}$ and $\sigma_{\bar{x}_1-\bar{x}_2}$.
        \vspace{0.6in}

  \item Find $P(\bar{x}_1 - \bar{x}_2 > 10)$.
        \vspace{0.8in}
\end{enumerate}
