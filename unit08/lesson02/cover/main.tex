\begin{learningtargetbox}
\begin{itemize}
    \item I can describe the shape, center, and spread of a chi-square distribution
          and explain how degrees of freedom affect its skewness.
    \item I can state H$_0$ and H$_a$ for a chi-square goodness of fit test, including
          all category proportions in the null hypothesis.
    \item I can calculate expected counts using $E = n \cdot p_0$ for each category.
    \item I can verify the Random and Large Counts ($E \ge 5$) conditions for a
          chi-square GOF test.
\end{itemize}
\end{learningtargetbox}
