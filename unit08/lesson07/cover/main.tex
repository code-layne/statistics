\begin{learningtargetbox}
\small
\begin{enumerate}[leftmargin=1.5em, itemsep=5pt]
  \item I can identify the correct inference procedure for a given categorical data scenario
        by recognizing whether the outcome is binary (two categories) or multi-category, and
        whether the data come from one sample or multiple groups.
        \textit{(VAR-6, UNC-4; Skill 1.E)}
  \item I can distinguish between z-procedures (one or two proportions) and chi-square
        procedures (goodness-of-fit, homogeneity, independence), and explain why each applies
        to its setting.
        \textit{(VAR-6, UNC-4; Skill 1.D)}
  \item I can state $H_0$ and $H_a$ in the correct parameter language for each of the five
        categorical inference procedures.
        \textit{(VAR-6, UNC-4; Skill 4.B)}
  \item I can identify the key condition to verify for a selected categorical inference
        procedure and state whether it is met given the data.
        \textit{(VAR-6, UNC-4; Skill 4.C)}
\end{enumerate}
\end{learningtargetbox}
