\begin{learningtargetbox}
\small
\begin{enumerate}[leftmargin=1.5em, itemsep=5pt]
  \item I can calculate the $\chi^2$ test statistic for a homogeneity or independence
        test using $\chi^2 = \displaystyle\sum \dfrac{(O-E)^2}{E}$.
        \textit{(VAR-8.L, Skill 3.E)}
  \item I can find the p-value using $\text{df} = (r-1)(c-1)$ and interpret it in
        context, explicitly assuming $H_0$ is true.
        \textit{(VAR-8.M / DAT-3.K, Skills 3.E, 4.B)}
  \item I can write a complete conclusion for a chi-square test for homogeneity or
        independence by comparing the p-value to $\alpha$ and answering the research
        question in context.
        \textit{(DAT-3.L, Skill 4.E)}
\end{enumerate}
\end{learningtargetbox}
