A quality-control engineer tests whether the mean fill volume of juice bottles differs
from the labeled amount of 12 fl oz. The hypotheses are $H_0{:}\ \mu = 12$ vs.\
$H_a{:}\ \mu \ne 12$, with $\alpha = 0.05$.

\begin{enumerate}

  \item Describe in context what it would mean to commit a \textbf{Type I error}.

        \writelines{2}

  \item Describe in context what it would mean to commit a \textbf{Type II error}.

        \writelines{2}

  \item The engineer decides to use $\alpha = 0.10$ instead of $\alpha = 0.05$.
        Describe the effect on each error rate.

        \begin{itemize}
            \item Type I error rate: \blank{5cm} (increases / decreases / stays the same)
            \item Type II error rate: \blank{5cm} (increases / decreases / stays the same)
        \end{itemize}

\end{enumerate}
