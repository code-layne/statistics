\begin{learningtargetbox}
\small
\begin{enumerate}[leftmargin=1.5em, itemsep=5pt]
  \item I can identify the correct inference procedure for a given scenario by determining
        the parameter type (proportion vs.\ mean) and the number of samples (one vs.\ two).
        \textit{(VAR-6, VAR-7; Skills 1.D, 1.E)}
  \item I can verify the appropriate conditions (Random, 10\%, Normal/Large Counts) for the
        selected inference procedure and explain in context whether each is met.
        \textit{(VAR-6, VAR-7; Skill 4.C)}
  \item I can carry out the selected inference procedure: compute the test statistic or
        interval and report the $p$-value or interval boundaries.
        \textit{(Skills 4.D)}
  \item I can communicate a conclusion in context using AP-exam language, linking the
        $p$-value (or CI) to the decision about the parameter.
        \textit{(Skills 3.D, 3.E, 4.E)}
\end{enumerate}
\end{learningtargetbox}
