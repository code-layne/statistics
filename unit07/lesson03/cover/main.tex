\begin{learningtargetbox}
\begin{itemize}
    \item I can correctly interpret a one-sample $t$-interval in context, referencing
          the specific sample and population.
    \item I can explain what ``$C$\% confident'' means in terms of repeated sampling and
          identify common misinterpretations.
    \item I can use a confidence interval to justify a claim about a population mean,
          determining whether a claimed value is plausible.
    \item I can identify the relationships between sample size, confidence level, and
          the width of a confidence interval for a mean.
\end{itemize}
\end{learningtargetbox}
