\begin{learningtargetbox}
\small
\begin{enumerate}[leftmargin=1.5em, itemsep=5pt]
  \item I can identify when to use a one-sample $t$-test (vs.\ a $z$-test) for a population
        mean, including when data come from matched pairs.
        \textit{(VAR-7.B, Skill 1.E)}
  \item I can state null and alternative hypotheses for a one-sample $t$-test or matched-pairs
        $t$-test in terms of $\mu$ (or $\mu_d$), choosing the correct direction for $H_a$.
        \textit{(VAR-7.C, Skill 1.F)}
  \item I can verify the Random/Independence and Normal conditions for a significance test
        about a population mean. \textit{(VAR-7.D, Skill 4.C)}
\end{enumerate}
\end{learningtargetbox}
