User is an AP Statistics teacher authoring a complete, multi-component lesson library. Each lesson has: cover, slides (Beamer), warmup, warmup_key, notes, notes_key, activity, activity_key, exit_ticket, exit_ticket_key, homework, homework_key.

Lessons are organized under `unit01/lesson02/`, etc. The project uses custom LaTeX packages (`apstats-article`, `apstats-beamer`, `apstats-key`, `apstats-boxes`) with shared color/style conventions (navy, sky, goldacc, greenacc, etc.).

User compiles PDFs themselves and reports layout issues back — Claude edits source, user compiles and checks.
